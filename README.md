# ✨ Beauty Filter — Prototype

ฟิลเตอร์กล้องบิวตี้แบบเรียลไทม์ ทำงานในเบราว์เซอร์ ฟรี 100% ไม่มีลายน้ำ

**เปิดใช้งานออนไลน์:** https://multithai.github.io/beauty-filter-prototype/

## ทำอะไรได้
- เกลี่ยผิวเนียนแบบ **bilateral filter** (เนียนแต่ยังเก็บรายละเอียดผิว ไม่เป็นพลาสติก)
- ปรับความสว่าง / โทนอุ่น / รักษารายละเอียดผิว
- เกลี่ยเฉพาะใบหน้า (ฉากหลังไม่โดน) โดยใช้ MediaPipe จับใบหน้า 468 จุด
- ปุ่มดูภาพ "ก่อน–หลัง"

## เทคโนโลยี
- **MediaPipe Face Landmarker** — ตรวจจับใบหน้า 468 จุดแบบ 3 มิติ
- **WebGL fragment shader** — ประมวลผลเอฟเฟกต์บน GPU แบบเรียลไทม์
- ประมวลผลในเครื่องผู้ใช้ทั้งหมด ไม่มีการส่งภาพออกไปเซิร์ฟเวอร์

## รันบนเครื่องตัวเอง
เปิดไฟล์ `index.html` ผ่านเว็บเซิร์ฟเวอร์ (กล้องต้องการ https หรือ localhost):
```
python -m http.server 8000
```
แล้วเปิด http://localhost:8000

> หมายเหตุ: ต้องใช้ Chrome/Edge/Firefox รุ่นใหม่ และอนุญาตสิทธิ์กล้อง
